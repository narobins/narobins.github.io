Autocorrect

Implementation of a prefix tree. Reads in text files, add words to tree, and generates suggestions by prefix matching, splitting the input, and calculating Levenshtein edit distance. Ranks suggestions based on similarity to input text and unigram/bigram frequencies. Java back-end; HTML/CSS/Javascript front-end.

KDTree

Implementation of a KDTree. Reads in csv file, stores 3D point coordinates of star locations, executes nearest-neighbor and radius searches for stars. Java back-end; HTML/CSS front-end.
